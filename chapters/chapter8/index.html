    Chapter: Quantum Phase Estimation (QPE)

    Quantum Phase Estimation (QPE) is a foundational quantum algorithm used to estimate the phase \( \phi \) in the eigenvalue \( e^{2\pi i \phi} \) of a unitary operator \( U \).

    üìò What is Quantum Phase Estimation?

    QPE estimates \( \phi \in [0,1) \) such that:

    \[ U|\psi\rangle = e^{2\pi i \phi}|\psi\rangle \]

    Where \( |\psi\rangle \) is an eigenvector of \( U \).

    üß† Why is Phase Estimation Important?

    QPE is essential in:

    • Shor's algorithm (period finding)
    • Quantum simulation (energy estimation)
    • Solving hidden subgroup problems

    üîó How Phase Helps with Period Finding

    In Shor‚Äôs algorithm, the goal is to find the period \( r \) of \( f(x) = a^x \bmod N \). The key insight is:

    • \( U|x\rangle = |a x \mod N\rangle \) has eigenstates with eigenvalues \( e^{2\pi i s/r} \)
    • QPE estimates the phase \( \phi = s/r \)
    • Use continued fractions to recover \( r \)

    ‚öôÔ∏è Circuit Diagram

    QPE uses:

    • \( t \) qubits for phase
    • 1 qubit for the eigenstate

    Then:

    1. Apply Hadamards to phase qubits
    2. Apply controlled-\( U^{2^j} \)
    3. Apply inverse QFT
    4. Measure

    from qiskit import QuantumCircuit, Aer, transpile, execute
from qiskit.circuit.library import QFT
import numpy as np
from qiskit.visualization import plot_histogram

# Accuracy (3 bits)
t = 3
phi = 0.125
shots = 1024

qc = QuantumCircuit(t + 1, t)
qc.h(range(t))
qc.x(t)  # Prepare |1‚ü© eigenstate of Rz

# Apply controlled-U^2^j gates
for j in range(t):
    angle = 2 * np.pi * phi * (2 ** j)
    qc.cp(angle, j, t)

# Apply inverse QFT
qc.append(QFT(num_qubits=t, inverse=True, do_swaps=True), range(t))
qc.measure(range(t), range(t))

# Simulate
sim = Aer.get_backend('aer_simulator')
qc = transpile(qc, sim)
result = execute(qc, backend=sim, shots=shots).result()
counts = result.get_counts()
plot_histogram(counts)

    Expected output: peak around 001 (i.e., \( 1/8 = 0.125 \)).

    üîó Role in Shor‚Äôs Algorithm

    Shor‚Äôs algorithm uses QPE to find \( r \), the period of \( a^x \bmod N \):

    • Define \( U |x\rangle = |a x \bmod N\rangle \)
    • QPE gives \( \phi = s/r \)
    • Use continued fractions to estimate \( r \)
    • Then compute factors using \( \gcd(a^{r/2} \pm 1, N) \)

    ‚úÖ Summary

    • Phase \( \phi \) is estimated by QPE
    • Helps recover the period in modular arithmetic
    • Essential to factoring large numbers (e.g., RSA)
    • Easily implemented in Qiskit for small \( \phi \)
